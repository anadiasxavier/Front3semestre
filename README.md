# Como rodar o dsGO?

## 1. Clonar o repositório
git clone git clone https://github.com/anadiasxavier/Front3semestre.git

## 2. Acessar a pasta do projeto
cd dsGo3_Aula1

## 3. Abrir no VS Code
code .

## 4. Instalar as dependências
npm i

## 5. Gerar a build de produção
npm run build

## 6. Rodar o projeto (preview da build)
npm run preview

O projeto dsGO roda através do comando:
npm run preview
